# Quesadillas

[https://www.youtube.com/shorts/pXcklWotRbo](https://www.youtube.com/shorts/pXcklWotRbo)

1. Fry base
    - spring onion
    - garlic
    - olive oil
2. Add mains to stir-fry
    - chopped mushrooms
    - paprika
3. Wrap: 
    - mushroom fry
    - cheese
    - parsley
4. Bake until crisp


+ Add cooked chicken to wrap for protein
