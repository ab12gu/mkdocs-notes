# Software

## Mac Commands

Update default app for code files (remove xcode... too slow):

```bash
$ brew install duti
$ osascript -e 'id of app "Zed"' # find bundle id for app
$ duti -s com.microsoft.VSCode .py all
$ duti -s dev.zed.Zed .js all
```
