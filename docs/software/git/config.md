# Config

Docs: [git-scm.com/docs/git-config](https://git-scm.com/docs/git-config)

The config file is the root of your terminal. The file is `~/.gitconfig`.

Add alias's to the file:

```
co = checkout
cm = commit
ss = status
b = branch
m = merge
a = add
```
