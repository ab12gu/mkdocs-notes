# Google Analytics



## Add to Material for Mkdocs

To add to [mkdocs](https://squidfunk.github.io/mkdocs-material/setup/setting-up-site-analytics/), just add these lines to `mkdocs.yaml`

```bash
extra:
  analytics:
    provider: google
    property: G-XXXXXXXXXX
```
