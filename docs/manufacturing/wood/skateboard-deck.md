# Skateboard Deck

## Steps

1. Buy thin plywood from hardware store
2. Buy wood glue
    - get type 2, water resistant
    - type 1, waterproof
3. Cut plywood and add glue
4. Clamp together
5. Repeat for each layer

Gorilla glue doesn't have wood types?
