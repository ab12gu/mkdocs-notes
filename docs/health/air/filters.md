# Filters

- N95 Masks
    - Particulate filtration (dust, sanding debris, some aerosols)
    - CANNOT protect against gas/fumes
- Surgical Masks
    - Protect spreading germs if your sick
- Gas/Respirator Masks
    - Depends on filter
        - Organic vapor cartridge
            - Protects against paint fumes, solvents, acetone, etc.
        - Particulate filters (P100/HEPA)A**
            - Protects against dust, metal particles, sanding debris.
        - Combination available


For car repair, best use P100 filters to protect brake dust (particles)

- NIOSH P100
- Brands: 3M, Honeywell
- lasts weeks to months
- [https://www.amazon.com/3M-65021HA1-C-Household-Multi-Purpose-Respirator/dp/B002NMFLU2?](https://www.amazon.com/3M-65021HA1-C-Household-Multi-Purpose-Respirator/dp/B002NMFLU2?)

If excessive use of brake cleaner/paint

- organic vapor cartridges (fumes)
    - paint thinner
    - acetone
    - degreasers

If sanding or using paint thinner + painting, need combo due to particles + fumes.
