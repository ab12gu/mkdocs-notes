# STM32


## STM32 Blue Pill

- Microcontroller: STM32F103C8T6 (Mainstream, Performance)
- Core: Arm Cortex-M3
- Started developing in 2007
- Max. Clock Speed	72MHz
- FLASH: 64KiB
- SRAM: 20KiB

### Power

- Any +3.3V pin (+3.3V)
- Any +5V pin (+5V)
- USB connector (+5V)
- Input: +3.6V to +5.5V
- Output: +3.3V @ 300mA
