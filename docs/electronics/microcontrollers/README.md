# Microcontrollers

It is a computer where all the components are on a single integrated circuit.

- Processing Cores 
- Memory
- Storage
- Input/Output terminals

![](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c8/Microcontrollers_Atmega32_Atmega8.jpg/1920px-Microcontrollers_Atmega32_Atmega8.jpg)

## Benefits

It is in a small form factor than a traditional computer/laptop. Most use significantly less power (nanowatts for an operation).
