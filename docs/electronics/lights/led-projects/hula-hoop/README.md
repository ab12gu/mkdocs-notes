## Materials for Custom Hoops

Materials

- 10ft polypropalene tubing
- smaller locking tubing
- gaffer tape
- rivets
- pop spring pin


Cutting Options:

- box cutter
- a hack saw
- miter saw
- chop saw
- tube cutters
