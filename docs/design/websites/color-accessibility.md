# Color Accessibility

**WCAG** stands for **Web Content Accessibility Guidelines**

- A: Basic accessibility.
- AA: Mid (most common target for websites)
    - Contrast Ratio Normal Text: 4.5:1
    - CR Large Text: 3:1
- AAA: Most strict
    - CR Normal Text: 7:1
    - CR Large Text: 4.5:1
